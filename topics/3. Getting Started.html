Getting Started

The follow articles describe the first steps to use the system, whether as an administrator or as an end user. For the administrator is necessary to get access to backend management system, providing a valid pair of user and password. One time inside the system the admin can add, remove or edit information like described in the section "Admin Interface Overview". AS for users, the step-by-step guide of the features of the system is proviede in section "User Interface Overview".

2.1 Prerequisites

Before proceeding with the ChameleonMap installation, it is essential that you have Docker and Docker Compose installed on your system. These tools are indispensable for running ChameleonMap.

Installing Docker on Linux

  1. Update your system's package index:

    sudo apt-get update

  2. Install packages to allow apt to use a repository over HTTPS:

    sudo apt-get install apt-transport-https ca-certificates curl software-properties-common

  3. Add Docker's official GPG key:

    curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

  4. Add the Docker repository to APT sources:

    sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

  5. Update the package index again after adding the Docker repository:

    sudo apt-get update

  6. Install Docker CE (Community Edition):

    sudo apt-get install docker-ce

  7. Verify that Docker was installed correctly:

    sudo docker run hello-world

    If the installation was successful, you will see a message indicating that Docker is running correctly.


Installing Docker Compose on Linux

  1. Download the latest version of Docker Compose:

    sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

  2. Make the Docker Compose binary executable:

    sudo chmod +x /usr/local/bin/docker-compose

  3. Verify the installation:

    docker-compose --version

    This should display the version of Docker Compose, confirming that the installation was successful.


After completing these steps, your system will be ready for the ChameleonMap installation. Proceed to the next section to learn how to configure and start ChameleonMap.

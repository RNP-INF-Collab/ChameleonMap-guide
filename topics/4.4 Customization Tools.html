Customization Tools

ChameleonMap is completely customized. In the section "Change Map Settings", in the admin page, there are a lot a parameters that can be modified by the system administrator:

1) Map Name:

The name of the map that appears on the web browser Tab title.

2) Map Style:

The admin can choose between two styles, "Gray Scale", which is the default, and "Blue colored", where the water bodies are presented in blue light color.

3) Inherit Child Location:

When this field is enabled, a tag that is parent from another tags will inherit all locations from the children that was not previously declared on the parent. If instead you want to manually declare the locations for all tags, we suggest to keep this field disabled.

4) Cluster Close Tags:

When this field is enabled, tags located very close to each other will be grouped in clusters. This mechanism only causes visual effects, not affecting the system behavior.

5) Initials zoom, latitude and longitude:

The initial observation point on the map.

6) Link feature:

When this field is enabled, the link feature will be displayed on the front-end.

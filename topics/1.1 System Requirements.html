Introduction to ChameleonMap

System Requirements

Server-side Requirements

  • Operating System: ChameleonMap server is compatible with Linux distributions.
  • Docker Composer and Docker: It is demand to have Docker and Docker Composer installed in server machine to run the system containers.
  • Hardware: A computer with a minimum of 32GB RAM and a dual-core processor is recommended for proper server service offering.

User-side Requirements

  • Web Browser: Ensure you are using the latest version of a modern web browser such as Google Chrome, Mozilla Firefox, or Safari.
  • Hardware: A computer with a minimum of 4GB RAM and a dual-core processor is recommended for smooth user-side operation.
  • Internet Connection: Users should have a stable internet connection to access and interact with ChameleonMap.

By meeting these system requirements, both on the server and user sides, you'll be ready to harness the full potential of ChameleonMap and explore its diverse applications in various domains.

Appendix

Frequently Asked Questions (FAQs)

1) What operational system do I need to instatiate the ChameleonMap system?

You can run the ChameleonMap system in any operation system, as long as it is capable to run docker and docker-compose application. However we recomend to use a linux-like OS.

2) Like an user, what web browser will I need to acess a ChameleonMap instatiated map?

ChameleonMap was projected to be compatible with the main web browsers, such as Mozila Firefox, Google Chrome and Microsoft Internet Explorer. Support for other browsers is not currently guaranteedAt the moment the support for other browsers is not guaranteed, as their correct operation has not been tested. But it can work.

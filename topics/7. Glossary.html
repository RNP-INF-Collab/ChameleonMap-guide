Appendix

Glossary

1) Tag:

An information unit. In the admin interface it is possible to add or edit tags, which one having a set of particular informations. Tags are grouped in tag menus and can be binded to locations. The same tag can be binded to multiples locations.

2) Location:

A specific geographic location on the global map, consisting of an latitude and a longitude. Locations can be added via the admin interface. It is possible to bind tags to each location, as many tags as desired.

3) Link:

A connection between two locations.

4) Tags Menu:

A palette of registered tags. A map instance can have many tags menus.

5) Links Menu:

A unique menu of links, that can be deactivated via admin interface.

6) ATLAS:

Acronym for Automated Topology Language with Slices, which is a topology representation language. In this context, ATLAS also refers to a representation system based in ATLAS language. The ChamaleonMap has an embedded ATLAS generator tool.
